
# React Candidates Assignment

Includes:

✔ React + TypeScript  
✔ Vite  
✔ MUI v5  
✔ TanStack Query  
✔ JSON Server  
✔ Reset Filters button  
✔ Unit tests (Vitest)

## Install dependencies (first thing)
npm install

## Run API (Server)
npm run server
- It uses the file **db.json** for data

## Run app (Client)
npm run dev

## Run tests
npm run test
